# About Task Tracker CLI

Task Tracker CLI is a lightweight, high-performance command-line application built in Go. It was created for developers and terminal power-users who want to manage their daily tasks without the overhead of heavy graphical interfaces or cloud-based applications.

## The Problem
Modern task management tools are often bloated. They require accounts, internet connections, and force you to click through multiple screens just to record a simple *to-do* item. 

## The Solution
We built Task Tracker CLI to eliminate that friction. It keeps you in the terminal—where you already work—allowing you to create, update, and manage tasks with a single line of text.

### Key Features
- **Instant Execution**: Compiled as a native Go binary, it runs immediately without starting a heavyweight runtime.
- **100% Local**: No databases to install or servers to connect to. Your data belongs to you and is stored safely in a local JSON file (`tasks.json`).
- **Zero Distractions**: A minimal text-based interface focused purely on productivity.
- **Status Management**: Quickly categorize tasks as `todo`, `in-progress`, or `done`.

## Built for the Terminal
If you use Git, Docker, or other standard Unix tools, Task Tracker CLI will feel right at home. It follows standard POSIX conventions, making it easy to integrate into shell scripts, aliases, or automation workflows.

Get started in seconds by keeping your focus on the code, not the task manager.
